# Task 10 – Animation of Multiple Wave Sources

## Problem Statement

Construct an HTML animation in which the user can place dots that act as sources of waves described by

$$
u(\vec{r},t) = \frac{A}{|\vec{r}-\vec{r_0}|^\alpha} \sin(k |\vec{r} - \vec{r_0}| - \omega t)
$$

where $\vec{r_0}$ is the position of a source and $\alpha$ is a parameter from the interval $[0,2]$. The animation must show the superposition of waves generated by all placed sources.

## Theory

A point source located at $\vec{r_0}$ produces a wave field that depends on the distance

$$
R = |\vec{r} - \vec{r_0}|
$$

The displacement generated by one source is

$$
u_i(\vec{r},t) = \frac{A}{R^\alpha} \sin(kR - \omega t)
$$

where:

- $A$ is the amplitude
- $k$ is the wave number
- $\omega$ is the angular frequency
- $\alpha$ controls the spatial decay of amplitude

The wave number is related to the wavelength by

$$
k = \frac{2\pi}{\lambda}
$$

If there are multiple sources, the total field is determined by the superposition principle:

$$
u_{\text{total}}(\vec{r},t) = \sum_i u_i(\vec{r},t)
$$

## Step-by-Step Solution

### Mathematical model

For every point of the simulation domain, the distance to each source is computed. The contribution from every source is then evaluated and summed.

### Adjustable parameters

The simulation includes adjustable parameters for:

- attenuation exponent $\alpha$
- wavelength $\lambda$
- angular frequency $\omega$
- amplitude $A$

### User interaction

The user places new wave sources by clicking on the canvas. Each click creates a point source at the selected position.

### Visualization

The total displacement field is visualized on a two-dimensional canvas. The color at each pixel represents the instantaneous value of the wave amplitude.

## Final Result

The simulation is implemented in a separate HTML file:

`task_10_animation.html`

## Interpretation

This model illustrates the superposition of circular wavefronts emitted by multiple sources. By changing $\alpha$, it is possible to investigate how the decay of amplitude with distance affects the global interference pattern.
